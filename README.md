# Maze Escape Game 🎮

## รายละเอียดเกม
เกมหนีรอดในเขาวงกต ที่ตัวละครของเราต้องวิ่งหนีปีศาจในเขาวงกตที่มีทางหนีให้ไล่ ปีศาจโหด ๆ ที่ไล่ตามเรา ตัวละครต้องวิ่งหนีปีศาจเพื่อไปเก็บเหรียญ 3 เหรียญให้ครบ พอครบแล้วก็จะจบด่านแล้วมีหน้าต่างให้ไปเลเวลถัดไป ความยากก็จะเพิ่มสูงตามเลเวล ยิ่งเลเวลสูงมีศาจก็จะมีเพิ่มขึ้น โหดขึ้น

## วิธีการเล่น
- ใช้ปุ่มลูกศรเพื่อเคลื่อนที่ตัวละคร
- หลีกเลี่ยงปีศาจสีแดง
- เก็บเหรียญสีเหลืองทั้งหมด 3 เหรียญเพื่อผ่านด่าน
- เมื่อผ่านด่าน กด SPACE เพื่อไปด่านถัดไป
- ถ้าถูกปีศาจจับได้ กด R เพื่อเริ่มใหม่

## วิธีการติดตั้งและรัน
1. ติดตั้ง Python 3.11 (เนื่องจาก Python 3.14 ยังไม่รองรับ Pygame อย่างสมบูรณ์)
2. สร้าง Virtual Environment (แนะนำ): `python -m venv venv`
3. Activate venv และติดตั้ง dependencies: `pip install -r requirements.txt`
4. รันเกม: `python main.py`
